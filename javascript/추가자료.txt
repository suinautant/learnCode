2. 입력 양식의 주요 속성
1) 읽기 전용 속성 : readonly
텍스트 상자에 쓰기를 제한하고 오직 읽기만 가능하게 설정 -> 쓰기 안됨
 
2) 비활성화 속성 : disabled
텍스트 상자를 비활성화 시킴 --> 값을 미리 정해놓고 입력 안되도록 함
 
3) 자동 완성 속성 : autocomplete
사용했던 데이터를 기준으로 입력 중인 텍스트에 자동 완성 기능을 적용할 수 있음

4) 자동 포커스 속성 : autofocus
현재 페이지가 로드될 때 처음으로 입력하고자 하는 폼을 선택하는 속성

5) 플레이스 홀더 속성 : placeholder
현재 폼에 입력해야 하는 텍스트를 희미하게 보여주는 속성
 
6) 필수 입력 속성 : required
반드시 데이터가 입력되어야 하는 폼을 지정하는 속성
 
7) 오타 체크 속성 : spellcheck
입력되는 문장의 오타를 실시간으로 점검


3. <datalist> 태그 --> 앞에서는 value가 보이지 않았는데 여기서는 value를 보여줌

텍스트 입력 시 자동완성 기능 제공



4. <optgroup> 태그

여러 항목을 그룹으로 묶을 수 있음


5. 막대 그래프 양식

<meter> 태그

지정된 값을 표현하기 위해 사용



<progress> 태그
